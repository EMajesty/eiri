# eiri

Atreus-inspired 40% keyboard with 6 columns

![eiri](https://imgur.com/ZiLTj3P.png)
![pcb](https://imgur.com/kHkkdv2.png)


## features

* Recessed Pro Micro for low profile
* 16 degree tilt per hand
